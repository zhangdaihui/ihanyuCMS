安装

  1. 将ihanyuCMS压缩包解压至一个空文件夹，并上传它。
  2. 首次在浏览器中访问index.php，将会进入安装向导。
  3. 按照安装向导完成安装。

最低系统需求

  • PHP 5.3.0 或更高版本。
  • MySQL 5.0 或更高版本。
