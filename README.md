# ihanyuCMS v1.0
ihanyuCMS是一个基于OneThink开发的内容管理框架。
### 新增
```sh
字段 : 颜色、图集、地图、天气
```
```sh
插件 : 分享到、友情链接、前台主题模板、返回顶部、CodeMirror编辑器
```
```sh
函数 : 关联独立模型
```
```sh
配置 : 架构保护、网站域名、统计代码、水印设置
```
```sh
功能 : 独立模型数据excel导出
```
### 修改
```sh
标签 : 推荐位(支持子分类)
```
```sh
插件 : 通用社交化评论(增加搜狐畅言)
```
```sh
函数 : is_administrator(支持多个超级管理员配置)
```
